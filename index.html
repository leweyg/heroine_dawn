Development

While looking for a fun game to write, I stumbled across Heroine Dusk, and like many others appreciated both the clarity of content and the clean code and data design. While trying to adapt it into a different game (Heroine Dawn), I felt that re-organizing the code to be more data centric would be appropraite, and so have exported most of the game content into a single JSON file.
